完成以霍夫直线检测为基准的刀具断开部分补全算法并在原图标记刀和伞。目前个人有两个补全算法的思路如下：
1.遍历图片（图片应经过单色调处理和降噪处理），找到每一行的起始断点，用直线连接该点与该行的最后一个点，若干直线构成连通区域。适用性比较强，但填补效果一般。
2.利用刀具和雨伞的形状特征，通过霍夫直线检测找到代表两种物体边缘的直线以及它们的交点，找到合适的四个交点并连接构成连通区域，补全空缺部分，填补效果还可以，但直线检测的质量取决于图片质量，变数较大。

物体分离算法架构：

物体提取：将完整的物体提取到新图片中。
